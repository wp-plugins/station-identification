== Description ==

Station Identification is an anti-spam/anti-feed-scraping plugin that allows the user to add a customizable copyright message at the bottom of all posts on their feeds. It can contain a link back to the original post, a link to the home page of the blog, and the author's name, and/or any arbitrary HTML the user desires. It is designed to be non-invasive for normal readers of your feed while ensuring that pirates and other seedy feedscrapers point readers back to the source of your content.

The default format is: "Post Title" was originally posted by Author Name on Blog Title. All rights are reserved by the author.

== Installation ==

Installing Station Identification is a breeze.

1. Upload the `station-identification` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all!
