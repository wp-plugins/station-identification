== Description ==

Station Identification is a plus that allows the user to add customizable copyright message at the foot of all posts on their feed. It contains a link back to the original post, a link to the home page of the blog, and the author's name.

It's default format is this: "Post Title" was originally posted by Author Name on Blog Title. All rights are reserved by the author.

== Installation ==

Installing Station Identification is a breeze.

1. Upload the `station-identification` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all!
